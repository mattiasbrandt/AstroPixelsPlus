Serial2 (Marcduino Input)

Recommended for protoArtoo body/dome link: 9600 baud, Body Link enabled, Serial2 ingest enabled.

JawaLite Protocol

When Body Link is enabled, firmware takes control of Serial2 for heartbeats and still forwards normal (non-heartbeat) commands to Marcduino handlers.
Not allowed: Body Link ON with Serial2 command ingest OFF. This page auto-forces Serial2 ingest ON while Body Link is enabled.

Body Controller Link

Bidirectional serial link protocol for integration with the Artoo Controller running the protoArtoo firmware.
Heartbeat flow: dome transmits #APHB and expects body #PAHB once per second.
Use this for dome/body sync over Serial2. If another device needs full control of Serial2, disable Body Link first.
